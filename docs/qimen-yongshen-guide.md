## 用神系统

用神是奇门遁甲断事的核心，根据不同事类选取相应的用神进行分析。

### 支持的事类

系统支持 **14 种事类**，每种事类有对应的主用神和辅用神：

| 事类 | 主用神 | 辅用神 | 说明 |
|:----:|:------|:------|:-----|
| 求财 | 生门、戊仪 | 开门、日干 | 财运、投资、经商 |
| 婚姻 | 六合、乙奇 | 开门、太阴 | 恋爱、婚配、感情 |
| 疾病 | 天芮、死门 | 日干、时干 | 健康、就医、康复 |
| 出行 | 日干、开门 | 马星、太冲 | 旅行、迁移、外出 |
| 诉讼 | 开门、庚仪 | 惊门、日干 | 官司、纠纷、争执 |
| 考试 | 丁奇、景门 | 天辅、日干 | 考学、面试、竞争 |
| 工作 | 开门、日干 | 生门、值符 | 事业、职位、升迁 |
| 失物 | 玄武、时干 | 六仪 | 寻物、追回、遗失 |
| 置业 | 生门、戊仪 | 天盘戊、日干 | 购房、投资、资产 |
| 求官 | 开门、值符 | 日干、景门 | 仕途、权位、晋升 |
| 孕产 | 天芮、六合 | 日干、太阴 | 怀孕、生育、母婴 |
| 寻人 | 时干、日干 | 开门 | 找人、联络、会面 |
| 合作 | 日干、时干 | 六合、开门 | 合伙、谈判、签约 |
| 其他 | 日干、时干 | 开门 | 通用类型 |

### 用神状态分析

每个用神会分析以下状态：

#### 1. 旺相休囚死

根据月令判断用神落宫的五行状态：

| 状态 | 含义 | 影响 |
|:----:|:-----|:-----|
| 旺 | 当令最旺 | +20分 |
| 相 | 次旺得生 | +10分 |
| 休 | 休息平和 | 0分 |
| 囚 | 受克被困 | -10分 |
| 死 | 最弱无力 | -15分 |

#### 2. 空亡检测

用神落入空亡宫位，力量减弱，事难成就。

#### 3. 入墓检测

用神落入墓库宫位，如木入未宫、火入戌宫等。

#### 4. 击刑检测

用神落宫与日支形成刑害关系。

#### 5. 与日干关系

| 关系 | 含义 | 影响 |
|:----:|:-----|:-----|
| 生 | 用神生日干 | +5分 |
| 比 | 同五行相助 | 0分 |
| 克 | 用神克日干 | -5分 |
| 泄 | 日干生用神 | -3分 |
| 耗 | 日干克用神 | -5分 |

### 主客分析

对于涉及双方的事类（如合作、诉讼），系统提供主客分析：

- **日干为我**（主方）
- **时干为彼**（客方）

分析内容包括：
- 双方落宫状态对比
- 五行生克关系（我克彼、彼克我、比和等）
- 综合判断结论

### 年命分析

可选提供求测者出生年干，分析年命落宫状态。

## 神煞系统

神煞是奇门盘中的吉凶标记，帮助判断时机的利弊。

### 支持的神煞

#### 吉神

| 神煞 | 查法 | 含义 |
|:----:|:-----|:-----|
| 天乙贵人 | 按日干查 | 最尊贵之神，主贵人相助、逢凶化吉 |
| 禄神 | 按日干查 | 主财禄、俸禄，利求财求官 |
| 天德 | 按月份查 | 最吉之神，百事皆宜 |
| 月德 | 按月份查 | 化凶为吉，有贵人暗助 |
| 天医 | 按月支查 | 主医药，利治病求医 |
| 丁马 | 丁奇落宫 | 主文书信息，利考试文章 |

#### 凶煞

| 神煞 | 查法 | 含义 |
|:----:|:-----|:-----|
| 羊刃 | 按日干查 | 主刚烈、冲动，凶煞之一 |
| 太岁 | 按年支查 | 岁君所在，宜静不宜动 |

#### 中性

| 神煞 | 查法 | 含义 |
|:----:|:-----|:-----|
| 驿马(日马) | 按日支查 | 主迁动变化，出行奔波 |

## 择日功能

择日功能帮助用户在指定日期范围内筛选吉时。

### 核心算法

择日采用综合评分系统：

```
总分 = 格局分×35% + 用神分×40% + 神煞分×25%
```

### 评级标准

| 评级 | 分数范围 | 建议 |
|:----:|:--------:|:-----|
| 优 | ≥80分 | 大吉，强烈推荐 |
| 良 | 65-79分 | 较宜，可以选用 |
| 中 | 50-64分 | 中平，酌情选用 |
| 差 | <50分 | 不宜，建议另择 |

### 过滤选项

| 选项 | 说明 |
|:-----|:-----|
| `minScore` | 最低评分阈值（默认60） |
| `limit` | 返回结果数量（默认10） |
| `excludeSuiPo` | 排除岁破日（太岁冲日） |
| `excludeYuePo` | 排除月破日（月建冲日） |
| `excludeJieQiDay` | 排除节气交接日 |
| `includeDirection` | 输出方位信息 |

### 方位信息

开启 `includeDirection` 后，输出包含：

1. **三吉门方位**：开门、休门、生门所在方位
2. **用神方位**：主用神所在方位

方位对应：

| 宫位 | 方位 |
|:----:|:----:|
| 坎一 | 北 |
| 坤二 | 西南 |
| 震三 | 东 |
| 巽四 | 东南 |
| 中五 | 中 |
| 乾六 | 西北 |
| 兑七 | 西 |
| 艮八 | 东北 |
| 离九 | 南 |

### 性能说明

择日功能采用 LRU 缓存优化：

| 范围 | 预期耗时 |
|:----:|:--------:|
| 7天 | < 50ms |
| 30天 | < 100ms |
| 365天 | < 2s |

## MCP 工具接口

### qimen_yongshen

奇门遁甲用神分析工具。

**输入参数**：

| 参数 | 类型 | 必填 | 说明 |
|:-----|:-----|:----:|:-----|
| year | number | 是 | 年份（1900-2100） |
| month | number | 是 | 月份（1-12） |
| day | number | 是 | 日期（1-31） |
| hour | number | 是 | 时辰（0-23） |
| minute | number | 否 | 分钟（0-59，默认0） |
| isLunar | boolean | 否 | 是否农历（默认false） |
| panType | string | 否 | 盘类型：时盘/日盘 |
| zhiRunMethod | string | 否 | 置闰法：chaibu/maoshan |
| shiLei | string | 是 | 事类（14种之一） |
| nianGan | string | 否 | 年干（用于年命分析） |
| includeShenSha | boolean | 否 | 是否包含神煞（默认true） |

**输出内容**：

- 完整奇门盘面（九宫、格局）
- 用神分析（主/辅用神、落宫、评分）
- 主客分析（如适用）
- 年命分析（如提供年干）
- 神煞信息（吉/凶/中性分组）

### qimen_zeri

奇门遁甲择日工具。

**输入参数**：

| 参数 | 类型 | 必填 | 说明 |
|:-----|:-----|:----:|:-----|
| startYear | number | 是 | 起始年份 |
| startMonth | number | 是 | 起始月份 |
| startDay | number | 是 | 起始日期 |
| endYear | number | 是 | 结束年份 |
| endMonth | number | 是 | 结束月份 |
| endDay | number | 是 | 结束日期 |
| shiLei | string | 是 | 事类（14种之一） |
| limit | number | 否 | 返回数量（默认10） |
| minScore | number | 否 | 最低评分（默认60） |
| includeDirection | boolean | 否 | 输出方位（默认false） |
| excludeSuiPo | boolean | 否 | 排除岁破 |
| excludeYuePo | boolean | 否 | 排除月破 |
| excludeJieQiDay | boolean | 否 | 排除节气日 |
| panType | string | 否 | 盘类型 |
| zhiRunMethod | string | 否 | 置闰法 |

**输出内容**：

- 推荐时辰概览表
- 详细分析（评分、格局、有利因素、注意事项）
- 方位信息（如开启）

## 使用示例

### 示例1：求财用神分析

```
事类：求财
时间：2025年1月15日 巳时

主用神：生门落兑七（相），戊仪落坤二（旺）
辅用神：开门落离九（休），日干甲落坎一（死）

分析：戊仪得位旺相，利求财；日干状态较弱，
需借助贵人。建议往西南方位求财。
```

### 示例2：出行择日

```
日期范围：2025年1月15日 - 1月20日
事类：出行
最低评分：70

推荐：
1. 1月17日 午时 - 86分（优）
2. 1月16日 巳时 - 81分（优）
3. 1月16日 丑时 - 77分（良）
```

---

## 注意事项

1. **用神优先级**：主用神权重高于辅用神
2. **空亡影响**：用神空亡时，即使其他条件好也需谨慎
3. **格局配合**：吉格配合吉门吉星为上
4. **择日范围**：建议不超过30天，以保证结果质量
5. **方位应用**：方位信息仅供参考，需结合实际情况
